# Momentum Strategy Scripts

All scripts related to the **Momentum Breakout** strategy.

## Files

| Script | Purpose |
|--------|---------|
| `backtest_momentum_5year.py` | 5-year momentum backtest |
| `backtest_momentum_options.py` | Momentum strategy with options |
| `backtest_momentum_portfolio.py` | Momentum portfolio backtest |
| `backtest_momentum_portfolio_v2.py` | Momentum portfolio v2 |
| `backtest_2year.py` | 2-year backtest variant |
| `full_backtest.py` | Full backtest suite |

## How to Run

```powershell
python "h:\QUANT TRADING\scripts\momentum\backtest_momentum_5year.py"
```

## Reports Output

Reports are saved to: `h:\QUANT TRADING\reports\momentum\`
